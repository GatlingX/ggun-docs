# Get Contract State (GET)

`GET https://api.sentou.ai/sentou-evm/contract/:address`

Retrieve the state of a contract

#### Request
This endpoint requires a path parameter.
- **address**: string (Required)

#### Response
This endpoint returns an object with the following fields:
- **stateVariables**: map from strings to strings
- **storage**: map from strings to strings
- **code**: string
- **balance**: integer
- **nonce**: integer
- **creator**: string


### Examples

```sh curl copy filename="POST /sentou-evm/contract/:address"
curl https://api.sentou.ai/sentou-evm/contract/:address \
     -H "X_API_KEY: <ApiKey>"
```
