# Quick Start

<Steps>
### Step 1

Install the Kalamari CLI

<Tabs items={['pip', 'pip3']}>
    <Tabs.Tab>
        ```sh pip copy
        pip install kalamari-cli
        ```
    </Tabs.Tab>
    <Tabs.Tab>
        ```sh pip3 copy
        pip3 install kalamari-cli
        ```
    </Tabs.Tab>
</Tabs>

### Step 2

<Callout type="warning" emoji="⚠️">
  You can ignore this step for now. This is for post-production
</Callout>

Get a Kalamari API key from the [Kalamari website](https://kalamari.com).

### Step 3

Set the `API_URL` environment variable to `http://34.89.77.42:8081` (Temporary)

```sh test copy
export API_URL=http://34.89.77.42:8081
```

### Step 4

Run the Kalamari CLI on a foundry project. `kalamari` expects a foundry project to be in the current directory, like below:
